## Projeto de IA Generativa com Microsoft Copilot

### 📝 Descrição do Projeto
Este projeto tem como foco a exploração dos recursos de Inteligência Artificial Generativa oferecidos pelo Microsoft Copilot, com ênfase na extração de texto a partir de imagens. A iniciativa demonstra como essa tecnologia pode ser utilizada para identificar e interpretar informações visuais de forma rápida, eficiente e precisa.

## 📁 Input
Imagens enviadas ao Copilot contendo textos a serem reconhecidos e descritos pela IA.

## 📁 Output
Resultados gerados pela IA após o processamento das imagens, contendo a descrição dos textos identificados nelas.
